# Bank Flag

## Deskripsi

Tim Problem Setter LKS telah menyusun sebuah dokumen penting yang dinamakan **Bank Flag**. Namun, akibat suatu kesalahan teknis, disk komputer LKS mengalami **corruption**. Tim Forensik ditugaskan untuk melakukan analisis terhadap disk image tersebut guna menemukan kembali data penting yang hilang.

Tugas kamu adalah melakukan **analisis forensik disk** untuk menemukan flag yang tersembunyi di dalam file sistem.
