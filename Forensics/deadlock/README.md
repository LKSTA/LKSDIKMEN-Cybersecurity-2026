# deadlock

## Deskripsi

Hash ini berasal dari artefak malware yang pernah berkomunikasi dengan infrastruktur internal LKS. Melalui platform **Threat Intelligence**, peserta ditugaskan untuk menelusuri **indikator kompromi (IoC)**, memetakan **infrastruktur Command and Control (C2)**, serta merekonstruksi pola aktivitas malware tersebut.

Tugas kamu adalah menganalisis informasi yang tersedia dan berinteraksi dengan layanan yang disediakan untuk menemukan flag.
